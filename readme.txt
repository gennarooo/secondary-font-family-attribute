Add font attribute to the text formatting toolbar in the block editor.

== Description ==

Add a new button to the block editor that add a <span> around the selected text with the class 'secondary-font-family'' in order to change font family within the same paragraph.

== Use ==

1. Install the plugin and activate.
2. Use the "Secondary font" toolbar button, available on any text content added in the block editor.
